# browser-extensions-fb
Browser extensions for focus blocked.

English:

Blocks websites from tracking when you switch tabs or lose focus on the browser window.

Disables JavaScript APIs (visibilitychange, blur, focus events) that websites use to monitor your activity.

Works on all websites by default, protecting your privacy automatically without configuration.

Unlocks text copying functionality even on websites that block it, allowing you to copy content with Ctrl+Alt.

Provides a blacklist feature to exclude specific domains where the extension should not work


Русский:

Блокирует отслеживание сайтами переключения вкладок и потери фокуса окна браузера.

Отключает JavaScript API (visibilitychange, blur, focus), которые сайты используют для мониторинга активности.

Работает на всех сайтах по умолчанию, автоматически защищая приватность без настройки.

Разблокирует копирование текста даже на сайтах, где это запрещено, позволяя копировать контент через Ctrl+Alt.

Предоставляет черный список для исключения определенных доменов, где расширение не должно работать.
